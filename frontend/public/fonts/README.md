# Fonts

This directory contains self-hosted font files used by the app to avoid build-time network fetches.

Fonts included:

- Geist (sans) — `geist.woff2`
- Geist Mono — `geist-mono.woff2`
- Playfair Display — `playfair-display.woff2`
- Space Grotesk — `space-grotesk.woff2`

These fonts are available via Google Fonts and are typically distributed under the SIL Open Font License (OFL).
